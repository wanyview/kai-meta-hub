> 设计目标: 完全本地化生成播客、视频、TTS、图片，不依赖外部服务

## 🎙️ 播客生成流程

### 输入
- Markdown 格式播客稿 (已有30+期)
- 格式: `**角色:** 对话内容`

### 处理
1. **解析**: 提取角色对话 + 场景描述
2. **角色分配**: 不同角色 → 不同音色
3. **TTS生成**: 逐段生成语音
4. **音频合成**: 背景音乐 + 对话 + 过渡音效

### 输出
- 完整 MP3/WAV 播客音频

### 角色音色示例

| 角色 | 音色风格 | 模型 |
|------|----------|------|
| Kai | 沉稳男声 | male-v2 |
| 范维友 | 专业女声 | female-v2 |
| 嘉宾A | 活力男声 | male-v3 |
| 嘉宾B | 温柔女声 | female-v3 |

---

## 🎬 视频生成流程

### 输入
- 播客音频
- 文章内容

### 处理
1. **文字转字幕**: 提取关键句子
2. **配图生成**: 根据内容生成/选取图片
3. **合成**: 字幕 + 图片 + 音频 → 视频

### 输出
- MP4 解说视频

---

## 🔊 TTS 朗读流程

### 输入
- 任意文本/文章

### 处理
1. **分段**: 按段落拆分
2. **生成**: TTS逐段生成
3. **合并**: 合成完整音频

### 输出
- MP3 语音文件

## 🚀 快速测试

```bash
# 1. 解析一期播客稿
python3 scripts/podcast_generator.py podcast_episode_01_fan_weiyou_final.md

# 2. 生成语音 (需要配置TTS)
./scripts/meta-podcast.sh --voice-kai male-deep podcast.md

# 3. 合成最终音频
ffmpeg -i part1.wav -i part2.wav -i music.wav -filter_complex amix=3 output.mp3
```
